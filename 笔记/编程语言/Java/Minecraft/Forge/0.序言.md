# Forge

>最常用的Minecraft模组框架

## 安装环境

1. 安装[对应版本](https://minecraft.fandom.com/zh/wiki/Java版#软件需求)的JDK
2. 在[Forge官网](https://files.minecraftforge.net/net/minecraftforge/forge/)下载MDK(Minecraft Development Kit)
3. 解压后用可以使用[Gradle](../../../构建工具/Gradle)的IDE打开，进行构建环境的自动配置
   + 不同网络环境配置时间差距**很大**，请保证**开启魔法**并在**良好的网络环境**下配置
4. `BUILD SUCCESSFUL`Gradle构建完成

## 项目配置

### 项目结构

```text
├── LICENSE
├── README
│
├── gradle
│   └── wrapper
│       ├── gradle-wrapper.jar
│       └── gradle-wrapper.properties
├── build.gradle
├── gradle.properties
├── gradlew
├── gradlew.bat
├── settings.gradle
│ 
└── src
    ├── generated
    │   └── resources
    ├── main
    │   ├── java
    │   │   └── net
    │   │       └── example
    │   │           └── mod
    │   │               └── ExampleMod.java
    │   └── resources
    │       └── META-INF
    │           └── mods.toml
    └── test
        └── java
            └── net
                └── example

```

#### 说明

##### build.gradle

`Mod Properties`需要设置

```properties
## Mod Properties
mod_id=examplemod #modid，唯一标识
mod_name=Example Mod #显示名称
mod_license=All Rights Reserved #开源协议
mod_version=1.0.0 #版本号
mod_group_id=com.example.examplemod #包名，唯一标识
mod_authors=YourNameHere, OtherNameHere #作者
mod_description=xxx.\nNewline xxx. #描述
```
